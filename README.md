## About the App

<p align="justify">Decofuture is an AR Furniture marketplace application that aims to bring furniture as close as possible to real life in your room before buying it, so you wont be dissapointed!
</p>

## Made With

- Flutter
- Google ARCore
- ModelViewer
- Firebase 

## Requirements

- Android 7.0+
